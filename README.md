Project Description:
This is an AI-powered Quiz App that generates quiz questions based on the topic entered by the user. 
The app uses the Gemini API to create questions and answers dynamically. 
It demonstrates API integration, JavaScript async functions, and DOM manipulation.

Features:
- Generate quiz questions using AI
- Enter any topic to get a question
- Shows both question and answer
- Simple and clean interface

  Technologies Used:
- HTML
- CSS
- JavaScript
- Gemini API
  
How It Works:
1. User enters a topic.
2. User clicks the Generate Quiz button.
3. The app sends a request to the Gemini API.
4. The API returns a question and answer.
5. The result is displayed on the page.
